Iniciando com HTML

O que é?

   HTML não é uma linguagem de programação, e sim uma linguagem de marcação de texto. Não se pode dizer "eu sou programador de HTML" ou "eu programo em HTML", HTML significa "Hyper Text Markup Language", em português, "Linguagem de marcação de hiper texto".

   O HTML é utilizado para estruturar sua página, esta estruturação na página permite você criar títulos, parágrafos, colocar imagens, colocar videos e envolver ou marcar diferentes partes do conteúdo, é exatamente como o próprio nome já diz "linguagem de marcação de texto".

   O HTML nos da infinitas possibilidades de estruturação, mas é importante saber que devemos seguir um "padrão" que ocorre e tentar ao máximo construir uma estrutura agradável ao usuário. Um exeplo de estruturação está neste próprio site, perceba que a navegação entre páginas foi colocada na parte superior do site e ela também pode ser colocada ao lado esquerdo, mas jamais deveria ser colocada ao lado direito ou embaixo (a não ser em casos extremamente específicos). Estruturar uma página em HTML irá demandar o conhecimento sobre "tags", também conhecidas como "elementos".

Tags / Elementos

   O HTML é utilizado para estruturar sua página, esta estruturação permite você criar títulos, parágrafos, colocar imagens, colocar videos, marcar conteúdos importantes e entre outras coisas, é exatamente como o próprio nome já diz "linguagem de marcação de texto".

   O HTML nos da infinitas possibilidades de estruturação, mas é importante saber que devemos seguir o "padrão" que ocorre na web, e tentar ao máximo construir uma estrutura agradável ao usuário. Um exeplo de estruturação está neste próprio site, perceba que a navegação entre páginas foi colocada na parte superior do site e ela também poderia colocada ao lado esquerdo, mas a navegação jamais deveria ser colocada ao lado direito ou embaixo (a não ser em casos extremamente específicos).

   Estruturar uma página em HTML irá demandar o conhecimento sobre as "tags", também conhecidas como "elementos" (uma tag é a mesma coisa que um elemento, diga de forma que preferir). Utilizamos das tags de HTML para formar a estrutura em si, seja a criação de títulos, botões, listas, tabelas, parágrafos.. Por exemplo, caso você queira colocar um título principal em sua página, você irá utilizar da tag "h1": 

  
 <body> 

    <h1> título principal! </h1> 

 </body>
Perceba que para criarmos um título foi necessário algumas coisas, tais como.:
  • Abrimento de tag: <h1>
  • Conteúdo da tag: título principal!
  • Fechamento da tag: </h1>

    Quando abrimos uma tag estamos informando ao HTML qual conteúdo desejamos criar, neste caso, o h1 foi colocado pois é a tag responsável por criar títulos. O nome "h1" não é algo aleatório, a letra "H" significa "heading", e de acordo com o dicionário online da Oxford, a palavra "heading" pode ser traduzida como um título impresso no topo de uma página ou no início de uma seção de um livro. Neste momento você pode se perguntar o motivo do número na tag, mas é um conceito bem simples. O número que acompanha a letra "H" é o level deste título, logo, quanto menor o level, mais importante este título é. Por exemplo a tag <h1> é o título principal da página (level 1), já o <h2> é o título dado a parágrafos, um título com menos importância (por isso é level 2). A variação de título vai do level um (h1) até o level seis (h6), sendo o "h1" e o "h2" os mais utilizados.

Atributos

   Como já foi dito anteriormente, utilizamos da tag para criar títulos, listas, tabelas. Porém as tags são utilizadas para CRIAR o conteúdo e não possuem capacidade de modificar o mesmo, e para isto servem os atributos HTML. Os atributos são palavras especiais usadas na abertura das tags, utilizamos para controlarmos o comportamento dessas tags. Por exemplo: 

  
 <body>   

    <h1 style="color: red;"> título principal! </h1> 
                       
 </body>
